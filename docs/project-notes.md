# Project Notes

This project starts with the frontend so the company workflow can be discussed visually before the backend schema and APIs are finalized.

## Initial Domain Ideas

- Employees or team members
- Departments or work groups
- Tasks and priorities
- Workload/capacity
- Assignment status
- Due dates and approvals

## Later Decisions

- Authentication and roles
- Task allocation rules
- Manual vs automatic assignment
- Reporting dashboard
- Notification channels
